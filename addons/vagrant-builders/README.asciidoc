Vagrant builders
================

Debian usage
------------

 cd debian/
 export GIT_BRANCH='feature-whatever'
 vagrant up

CAUTION: GIT_BRANCH should be 'feature-whatever', not 'feature/whatever'

CentOS usage
------------

 cd centos/
 export GIT_BRANCH='feature/whatever'
 export PFREPO='whatever'
 vagrant up
